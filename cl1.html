CSS Lesson 1 : Introduction to CSS

CSS, or Cascading Style Sheets, is a style sheet language used to describe the presentation of a document written in HTML. CSS allows web developers to control the layout, colors, fonts, and other visual aspects of a webpage, separate from its structure and content.


1. Selectors

  • Selectors are patterns used to select the HTML elements you want to style.
  • They can target elements based on their tag name, class, ID, or other attributes.
  • For example, to select all paragraphs, you would use the selector p. To select elements with a specific class, you would use .classname.

2. Properties

  • CSS properties are the styling attributes that you apply to selected elements.
  • Properties define how elements should appear, such as their color, size, font, margin, padding, and more.

3. Values

  • CSS properties are assigned values that determine the specific styling applied to elements.
  • Values can be keywords, such as red or bold, or numerical values, such as 12px or 2em.

4. Syntax

  • CSS rules consist of a selector, followed by a block of property-value pairs enclosed in curly braces { }.
  • Each property-value pair is separated by a colon :, and individual rules are separated by semicolons ;.

5. Cascading

  • CSS follows a cascading model, meaning that multiple style sheets can be applied to the same document, and styles can cascade or override each other based on specificity and order of declaration.

Applying CSS to HTML

1. Inline Styles

Styles can be applied directly to individual HTML elements using the style attribute.

<p style="color: blue; font-size: 16px;"> 
This is a styled paragraph.</p>

2. Internal Styles

Styles can be defined within the <style> element in the <head> section of an HTML document.

<head>
    <style>
        p {
            color: blue;
            font-size: 16px;
        }
    </style>
</head>

3. External Stylesheets

Styles can be defined in external CSS files and linked to HTML documents using the <link> element.

<head>
    <link rel="stylesheet" href="styles.css">
</head>
