HTML Lesson 1: Introduction to HTML

HyperText Markup Language (HTML). It's the standard language used to create web pages. HTML defines the structure and layout of a web page by using a variety of tags and attributes.


Elements

HTML documents are made up of elements, which are the building blocks of a webpage. Elements are defined by HTML tags enclosed in angle brackets (< >). Each element serves a specific purpose, such as defining paragraphs, headings, links, images, and more.

Tags

Tags are used to mark up the content within an HTML document and provide structure to the page. They come in pairs: an opening tag and a closing tag. The opening tag contains the name of the element, while the closing tag contains a forward slash (/) before the element name. For example, <p> is the opening tag for a paragraph, and </p> is the closing tag.

Attributes

HTML elements can have attributes, which provide additional information about an element. Attributes are specified within the opening tag and are written as name-value pairs. For example, the href attribute in an anchor tag (<a>) specifies the URL of the link.

Structure

HTML documents have a hierarchical structure, with elements nested within other elements. The <html> element is the root element of an HTML document and contains all other elements. Inside <html>, there are <head> and <body> elements. The <head> contains meta-information about the document, such as the title and links to external resources, while the <body> contains the visible content of the page.

Text Content

HTML allows you to include various types of content within your web pages, such as text, images, videos, links, and more. Text content is usually enclosed within appropriate tags, such as <p> for paragraphs, <h1> to <h6> for headings, <a> for links, and so on.
