CSS Lesson 3 Quiz

1.What does the CSS selector div p target?

  • A) All <div> elements
  • B) All <p> elements
  • C) All <p> elements inside <div> elements
  • D) All <div> elements inside <p> elements

Correct!

Incorrect! The correct answer is C.

2.Which CSS selector targets all <li> elements that are direct children of <ul> elements?

  • A) ul li
  • B) ul + li
  • C) ul > li
  • D) ul ~ li

Correct!

Incorrect! The correct answer is C.

3.What are pseudo-classes and pseudo-elements used for in CSS?

  • A) Defining special states or parts of elements
  • B) Adding new elements to the DOM
  • C) Styling elements based on their IDs
  • D) Creating animations and transitions

Correct!

Incorrect! The correct answer is A.

4.What are the components of the CSS box model?

  • A) Content, margin, padding, border
  • B) Margin, border, padding, content
  • C) Content, padding, border, margin
  • D) Padding, content, margin, border

Correct!

Incorrect! The correct answer is A.

5.What does the box-sizing property control?

  • A) The shape of the box model
  • B) The behavior of elements within a box
  • C) The calculation of width and height of an element
  • D) The position of the box within its parent

Correct!

Incorrect! The correct answer is C.

6.How does relative positioning move an element?

  • A) Relative to its parent element
  • B) Relative to the viewport
  • C) Relative to its siblings
  • D) Relative to its normal position in the document flow

Correct!

Incorrect! The correct answer is A.

7.When using absolute positioning, what does an element position itself relative to?

  • A) Its parent element
  • B) Its nearest positioned ancestor
  • C) The viewport
  • D) The document body

Correct!

Incorrect! The correct answer is C.

8.Which layout model allows you to create flexible and responsive layouts with a container and its child items?

  • A) Flexbox
  • B) Grid Layout
  • C) Relative Positioning
  • D) Absolute Positioning

Correct!

Incorrect! The correct answer is D.

9. What do media queries allow you to do in CSS?

  • A) Change the font size of elements
  • B) Apply different styles based on device characteristics
  • C) Create animations based on user interactions
  • D) Define the structure of the HTML document

Correct!

Incorrect! The correct answer is B.

10.What is the purpose of the <meta name="viewport"> tag in HTML?

  • A) To specify the font size for mobile devices
  • B) To control the behavior and scale of the viewport on mobile devices
  • C) To include external CSS files
  • D) To define the character encoding of the document

Correct!

Incorrect! The correct answer is A.
