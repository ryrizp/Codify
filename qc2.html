CSS Lesson 2 Quiz

1.What are the three main components of CSS syntax?

  • A) Selectors, Tags, Attributes
  • B) Selectors, Properties, Values
  • C) HTML, JavaScript, Properties
  • D) Classes, IDs, Elements

Correct!

Incorrect! The correct answer is B.

2.Which part of CSS specifies the aspect of the element you want to change?

  • A) Selector
  • B) Property
  • C) Value
  • D) Attribute

Correct!

Incorrect! The correct answer is B.

3.How do you apply styles directly to individual HTML elements?

  • A) Inline Styles
  • B) Internal Styles
  • C) External Stylesheets
  • D) JavaScript Styles

Correct!

Incorrect! The correct answer is A.

4.Where do you define styles within an HTML document using the <style> element?

  • A) Inside the <llbody> tag
  • B) Inside the <head> tag
  • C) Inside the <footer> tag
  • D) Inside the <nav> tag

Correct!

Incorrect! The correct answer is B.

5.Which method of applying styles to HTML elements allows for centralized styling?

  • A) Inline Styles
  • B) Internal Styles
  • C) External Stylesheets
  • D) JavaScript Styles

Correct!

Incorrect! The correct answer is C.

6.What HTML attribute is used to link an external CSS file to an HTML document?

  • A) src
  • B) href
  • C) style
  • D) link

Correct!

Incorrect! The correct answer is B.

7.In the example <p style="color: blue; font-size: 16px;"">, how are the styles applied to the paragraph element?

  • A) Inline Styles
  • B) Internal Styles
  • C) External Stylesheets
  • D) JavaScript Styles

Correct!

Incorrect! The correct answer is A.

8.What is the purpose of using External Stylesheets?

  • A) To apply styles directly to individual HTML elements
  • B) To define styles within the <style> element
  • C) To link styles from an external CSS file to an HTML document
  • D) To dynamically change styles using JavaScript

Correct!

Incorrect! The correct answer is C.

9. Where do you typically place the <style> element in an HTML document?

  • A) Before the opening <html> tag
  • B) After the closing </html> tag
  • C) Inside the <body> tag
  • D) Inside the <head> tag

Correct!

Incorrect! The correct answer is D.

10.Which element in CSS syntax selects the HTML element(s) you want to style?

  • A) Selector
  • B) Property
  • C) Value
  • D) Attribute

Correct!

Incorrect! The correct answer is A.
